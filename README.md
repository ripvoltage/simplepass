# simplepass

VERY simple password generator

# Usage

First, make sure to compile it using GCC or any preferred compiler.

```
gcc simplepass.c -o simplepass
```

It does not require any external devel libraries.

Then, simply run:

```
./simplepass [Amount of characters]
./simplepass 16
```

This will generate a random string of 16 characters that you can then use as a password. It will be printed in the terminal, so make sure no one is watching. You can clear the terminal after securely saving it.
